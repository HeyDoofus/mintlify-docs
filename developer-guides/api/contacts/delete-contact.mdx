# Delete a contact

Remove a contact from a chatbot.

## Base URL

```bash
DELETE https://api.doofus.com/v1/chatbots/{id}/contacts/{contactId}
```

## Authentication

```bash
Authorization: Bearer YOUR_API_KEY
```

## Example Request

```bash
curl -X DELETE https://api.doofus.com/v1/chatbots/bot_abc123/contacts/user_1 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json Response (204)
{ }
```
