# Get a specific contact

Retrieve details for a single contact.

## Base URL

```bash
GET https://api.doofus.com/v1/chatbots/{id}/contacts/{contactId}
```

## Authentication

```bash
Authorization: Bearer YOUR_API_KEY
```

## Example Request

```bash
curl -X GET https://api.doofus.com/v1/chatbots/bot_abc123/contacts/user_1 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json Response (200)
{ "contactId": "user_1", "email": "user@example.com", "name": "Jane" }
```

```json Response (404)
{ "error": "NotFound", "message": "Contact not found" }
```
