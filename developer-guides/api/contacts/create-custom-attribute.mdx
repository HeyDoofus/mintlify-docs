# Create custom attribute

Create a new custom attribute for contacts.

## Base URL

```bash
POST https://api.doofus.com/v1/chatbots/{id}/contacts/attributes
```

## Authentication

```bash
Authorization: Bearer YOUR_API_KEY
```

## Request Body

- `name` (string) - attribute name
- `type` (string) - attribute type (string, boolean, number)

## Example Request

```bash
curl -X POST https://api.doofus.com/v1/chatbots/bot_abc123/contacts/attributes \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "name": "vip", "type": "boolean" }'
```

```json Response (201)
{ "name": "vip", "type": "boolean" }
```
