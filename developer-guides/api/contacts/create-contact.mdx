# Create contact

Create a new contact associated with a chatbot.

## Base URL

```bash
POST https://api.doofus.com/v1/chatbots/{id}/contacts
```

## Authentication

```bash
Authorization: Bearer YOUR_API_KEY
```

## Request Body

- `email` (string) - Contact email
- `name` (string)
- `externalId` (string)

## Example Request

```bash
curl -X POST https://api.doofus.com/v1/chatbots/bot_abc123/contacts \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "email": "user@example.com", "name": "Jane" }'
```

```json Response (201)
{ "contactId": "user_1", "email": "user@example.com", "name": "Jane" }
```
