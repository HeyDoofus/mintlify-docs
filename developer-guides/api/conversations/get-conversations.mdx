# Get conversations for a chatbot

Retrieve conversations for a specific chatbot.

## Base URL

```bash
GET https://api.doofus.com/v1/chatbots/{id}/conversations
```

## Authentication

```bash
Authorization: Bearer YOUR_API_KEY
```

## Query Parameters

- `page` (integer)
- `limit` (integer)

## Example Request

```bash
curl -X GET "https://api.doofus.com/v1/chatbots/bot_abc123/conversations?page=1&limit=20" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json Response (200)
{
  "data": [ { "conversationId": "conv_123", "lastMessage": "Hi" } ],
  "meta": { "page": 1, "limit": 20 }
}
```

```json Response (404)
{ "error": "NotFound", "message": "Chatbot not found" }
```
