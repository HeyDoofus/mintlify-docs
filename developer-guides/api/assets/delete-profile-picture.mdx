# Delete chatbot profile picture

Remove the chatbot's profile picture.

## Base URL

```bash
DELETE https://api.doofus.com/v1/chatbots/{id}/assets/profile-picture
```

## Authentication

```bash
Authorization: Bearer YOUR_API_KEY
```

## Example Request

```bash
curl -X DELETE https://api.doofus.com/v1/chatbots/bot_abc123/assets/profile-picture \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json Response (204)
{ }
```
