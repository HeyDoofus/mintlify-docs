# Upload chatbot profile picture

Upload a profile picture for the chatbot.

## Base URL

```bash
POST https://api.doofus.com/v1/chatbots/{id}/assets/profile-picture
```

## Authentication

```bash
Authorization: Bearer YOUR_API_KEY
```

## Example Request

```bash
curl -X POST https://api.doofus.com/v1/chatbots/bot_abc123/assets/profile-picture \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@./profile.jpg"
```

```json Response (201)
{ "url": "https://cdn.doofus.com/assets/profile_bot_abc123.jpg" }
```
