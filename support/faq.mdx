# Frequently Asked Questions

## General Questions

### What is Doofus?
Doofus is an AI agent platform that allows you to create intelligent assistants trained on your business data.

### How does Doofus work?
Doofus uses advanced language models combined with your custom data to create AI agents that can understand and respond to user queries in a context-aware manner.

### What can I build with Doofus?
- Customer service chatbots
- Knowledge base assistants
- Sales support agents
- Technical support bots
- Lead generation agents
- Data analysis assistants

### How do I handle rate limits?
```typescript
// Example of handling rate limits
try {
  const response = await doofus.chat.create({
    message: "Hello",
    retry: true,
    maxRetries: 3,
    retryDelay: 1000
  });
} catch (error) {
  if (error.code === 'rate_limit_exceeded') {
    // Handle rate limit
    await delay(error.retryAfter);
  }
}
```
