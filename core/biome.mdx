## What is Biome?

Biome is a modern JavaScript/TypeScript toolchain that handles both linting and formatting. It replaces ESLint and Prettier in our project setup.

## Configuration

The Biome configuration is stored in `biome.json`:

```json
{
  "$schema": "https://biomejs.dev/schemas/1.0.0/schema.json",
  "organizeImports": {
    "enabled": true
  },
  "linter": {
    "enabled": true,
    "rules": {
      "recommended": true
    }
  },
  "formatter": {
    "enabled": true,
    "indentStyle": "space",
    "indentWidth": 2
  }
}
```

## Usage

### Linting
Run the linter using:
```bash
npm run lint
# or directly
biome check .
```

### Formatting
Format your code using:
```bash
npm run format
# or directly
biome format --write
```

## VS Code Integration

For the best development experience, install the Biome VS Code extension. It provides:
- Real-time linting
- Format on save
- Quick fixes
- Import organization

## Common Rules

Biome enforces several key rules:
- Consistent quotation marks
- No unused variables
- Proper spacing
- Import organization
